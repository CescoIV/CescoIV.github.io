Here is some of my work

Study Nahuatl

Application built to help people study the Nahuatl language as well as to provide a Front-End solution for its API. Built with ReactJS.

Nahuatl API

Back-End Solution to The Nahuatl Project, REST API built with MongoDB and NodeJS. Instructions for utilization in the github Readme.

My Travels

Team Project solution built to help travelers keep track of the cities/countries they have visited. I contributed in our oAuth implementation as well as in the Back-End NodeJS source and the logo design

Food Dice

Team Project solution built for indecisive individuals or people new to a town looking for food. Built in ReactJS, contributed to the project's architecture and API mechanics, as well as several of the actual components.

Pi-Tac-Toe

My first programming project, a take on a pirate themed tic-tac-toe game with a small twist of mayhem and custom made graphics. Not yet deployed(tentative).
